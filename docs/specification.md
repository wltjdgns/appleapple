# 사과게임(Apple Game) 프로젝트 설계 명세서

## 1. 개요
웹 브라우저에서 실행되는 사과게임으로, 숫자 합이 10인 영역을 제거하는 정통 퍼즐 게임입니다.

## 2. 게임 모드
- **일반 모드**: 120초 제한 시간 내 고득점 목표 (17x10).
- **멀티 모드**: 로컬 1:1 대결 또는 향후 확장 가능한 멀티플레이 구조.
- **특수 모드**:
  - **무한 시간**: 시간 제한 없음.
  - **거대 모드**: 50x30 대형 그리드 (성능 최적화 반영).
  - **퍼펙트 시드**: 100% 제거 가능한 특수 알고리즘 적용.

## 3. 핵심 메커니즘
- **그리드 시스템**: 동적 행/열 지원 (기본 17x10).
- **영역 합계 로직 (SAT)**: `Summed-Area Table`을 활용하여 드래그 영역 합을 $O(1)$로 계산.
- **제거 및 점수**: 합이 10인 영역 제거, 사과 1개당 1점.
- **상태 관리**: 보드 데이터와 점수 동기화.

## 4. 알고리즘: 퍼펙트 시드 생성 (n-Sum Partitioning)
1. 보드를 무작위 크기의 직사각형 영역들로 완전히 분할.
2. 각 영역의 합이 10이 되도록 숫자를 랜덤하게 채움 (2-sum ~ n-sum 혼합).

## 5. 기술 스택
- **Engine**: Pure JavaScript 클래스.
- **UI**: HTML5 Canvas, Vanilla CSS/JS.
- **Hosting**: GitHub Pages.

## 6. 파일 구조
- `/core/engine.js`: GameEngine 클래스.
- `/core/generator.js`: SeedGenerator 클래스.
- `/web/app.js`: 웹 UI 컨트롤러 및 렌더러.
- `/web/index.html`: 메인 게임 화면.
